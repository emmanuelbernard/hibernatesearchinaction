Chapter 8 - Hibernate Search in Action

This chapter covers dynamic filtering.

Project structure

src

Contains the example codes. The action package contains the examples using Filters in action. the model package and more precisely model.Item contains the filter declarations. And action.filter contains the filter and filter factory code.

Examples are split between DistributorAction, ItemRetrievalAction and StockAction.

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

Each test case make use of a data set contained in /import.sql. This data set is a sample of a realistic online catalog.

Each test use a proxy based session/entitymanager initialization and transaction opening/closing. Each test runs inside a transaction just like it would in a container. SessionHolder and EntityManagerHolder keep the FullTextSession and FullTextEntityManager in a thread local for consumption by the services.

Dependencies

You can find the libraries necessary to compile and run the tests in ../common/lib
