Hibernate Search in Action - companion code

This document explains how to use the companion code of Hibernate Search in Action

This description is relevant for chapter 2 to 6 and 8 to 10

Project structure

src

Contains the example codes.

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

Dependencies

You can find the libraries necessary to compile and run the tests in common/lib

Build

Each project contains an Ant based build system. The following tasks are available:

  • clean: clean resources
  • compile: compile the source code
  • test: run the TestNG tests. Results can be found in build/reports.

This build has been tested with Ant 1.7 and JDK 1.5

Eclipse users

You can import each project in your workspace. Open a workspace pointing to the root directory. File -> import -> General -> Existing Projects into Workspace. Select the root directory as the code root directory (directory containing this file). Import all projects (including common).

This has been tested on Eclipse Europa. Using the TestNG plugin allows you to run TestNG tests from your IDE.

IntelliJ users

Open HSiA-DVDStore.ipr thout you can found in the root directory. Make sure that in preferences -> compiler the following resource patterns are present ?*.properties;?*.xml;?*.dtd;?*.txt;?*.sql . It should. Enjoy.

This procedure has been tested on IntelliJ 7.0.4. Using the TestNG plugin allows you to run TestNG tests from your IDE.
