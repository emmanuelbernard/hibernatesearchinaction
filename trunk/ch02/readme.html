Chapter 2 - Hibernate Search in Action

This chapter is an introduction to Hibernate Search. It describes how to configure it and how to use the most basic APIs

Project structure

src

Contains the example codes. The package model contains the domain model and demonstrate the use of mapping annotations. The package action contains examples of the Hibernate Search API usage. Also contains the configuration files.

  • Example 2.3: Shows typical configurations mechanism. See /hibernate.cfg.xml and META-INF/persistemce.xml
  • Example 2.5: Shows a configuration of Hibernate Search without Hibernate Annotations. Event listeners must be registered. See /hibernate-no-annotation.cfg.xml
  • Example 2.6: describes annotation based mappings. See model/Item.java
  • Example 2.7: shows how Hibernate Search and hbm.xml files can be used together. See model/Item.hbm.xml
  • Example 2.8: shows how to access the main Hibernate Search API FullTextSession or FullTextEntityManager. See action/Indexer.java
  • Example 2.9: shows how to manually index entities.
  • Example 2.10: string based Lucene query. See action/Searcher#search()
  • Example 2.11: string based Lucene query using the MultiFieldQueryParser. See action/Searcher#searchMultipleFields()
  • Example 2.12: Building an Hibernate Search query. See action/Searcher#searchMultipleFields()
  • Example 2.13: Execute an Hibernate Search query. See action/Searcher#searchMultipleFields()

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

Dependencies

You can find the libraries necessary to compile and run the tests in ../common/lib
