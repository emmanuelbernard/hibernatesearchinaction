Chapter 5 - Hibernate Search in Action

This chapter covers directory providers, analyzer definitions and indexing.

Project structure

src

Contains the example codes. The package model contains the domain model and demonstrate the use of analyzers. action shows analyzers in action and how to manually index your data

Item hosts most of the analyzer definition examples and their usage on entities and fields. An example of stop word file is shown in com/manning/hsia/dvdstore/stopwords.txt. StemmerIndexerImpl tests the behavior of the stemmer analyzer and its influence on queries.

Indexer in the action package demonstrates how to do manual indexing

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

Each test case make use of a data set contained in /import.sql. This data set is a sample of a realistic online catalog.

Dependencies

You can find the libraries necessary to compile and run the tests in ../common/lib
