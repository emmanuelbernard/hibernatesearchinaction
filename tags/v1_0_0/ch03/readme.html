Chapter 3 - Hibernate Search in Action

This chapter covers basic mapping annotations.

Project structure

src

Contains the example codes. The package model contains the domain model and demonstrate the use of mapping annotations. Also contains the configuration files.

This project represents the same code base as described in the book but does not cover all cases described in Chapter 3. It gives a good sample of annotation usage. example.hibernate.cfg.xml does describe a configuration where Lucene indexes are stored on the file system (see Example 3.7).

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

Dependencies

You can find the libraries necessary to compile and run the tests in ../common/lib
