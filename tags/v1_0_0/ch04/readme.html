Chapter 4 - Hibernate Search in Action

This chapter covers advanced mapping such as custom bridges and association indexing.

Project structure

src

Contains the example codes. The package model contains the domain model and demonstrate the use of mapping annotations.

model/Item contains most of the mapping examples. Bridges are present in the bridge package. Association mapping are described in Item and its associated entities.

test

Contains the test framework. Several tests are making sure the examples runs. The test framework uses TestNG.

BridgeTest is a unit test showing how bridges react. BridgeUseTest shows bridges within the Hibernate Search infrastructure. IndexedEmbeddedTest contains all the association mapping tests

Dependencies

You can find the libraries necessary to compile and run the tests in ../common/lib
